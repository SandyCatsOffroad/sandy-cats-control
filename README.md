# Sandy Cats Mission Control

Business management dashboard for Sandy Cats manufacturing operations.

## Features

- 📋 **Tasks** - General business task management
- 📅 **Calendar** - Events, deadlines, and meetings
- 🔍 **SEO Tasks** - Keyword tracking and content planning
- 📢 **Ads** - Campaign performance and budget tracking
- 📱 **Social Media** - Post scheduling and engagement metrics

## Access

Live dashboard: https://YOUR_GITHUB_USERNAME.github.io/sandy-cats-control

## Data Management

All data is stored in JSON files under `/data/`:
- `tasks.json` - Business tasks
- `calendar.json` - Calendar events
- `seo.json` - SEO tasks and metrics
- `ads.json` - Ad campaigns
- `social.json` - Social media posts

Update these files and commit to update the dashboard remotely.

## Local Development

1. Open `index.html` in a browser
2. Dashboard auto-refreshes data every 60 seconds
3. Works offline after initial load

## Deployment

Deployed via GitHub Pages. Any push to `main` branch updates the live site.
